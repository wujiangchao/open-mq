# open-mq

| 模块名              | 说明                       |
| ------------------- | -------------------------- |
| broker      | 消息存储、消息转发       |
| store      | 消息持久化        |
| client    | producer、consume都属于client         |
| controller    | 负责Broker Master的选举和通知Broker|
| namesrv      |元数据存储，原zk角色         |
| remoting      | 远程通信模块       |
| proxy   | Grpc proxy      |
| common | 通用能力、工具|
